## 3D
*3D-centric Libraries*

* [anim9 ★17](https://github.com/excessive/anim9) - 3D skeletal animation library (design to be used with IQM and IQE)
* [bump-3dpd ★12](https://github.com/oniietzschan/bump-3dpd) - A 3D collision detection library for Lua.
* [DEEP ★3](https://github.com/Nikaoto/DEEP) - Handles draw order by allowing you to specify Z axis when drawing.  
* [IQE ★10 ⏳2Y](https://github.com/excessive/iqe) - Inter-Quake Export loader (text)
* [IQM ★18 ⏳1Y](https://github.com/excessive/iqm) - Inter-Quake Model loader (binary)
* [LÖVE3D ★139](https://github.com/excessive/love3d) - 3D extensions to LÖVE
* [Lovox ★25 ⏳1Y](https://github.com/tjakka5/Lovox) - Pseudo-3D library for working with voxels

*Navigation, Decision-Making and AI Libraries*

* [Jumper ★416](https://github.com/Yonaba/Jumper) - Grid-based pathfinding library

## Animation

* [anim8 ★273](https://github.com/kikito/anim8) - Animation library
* [chiro ★10](https://github.com/bjornbytes/chiro) - Convenience wrapper around [Spine](http://esotericsoftware.com)
* [skeletor ★12 ⏳2Y](https://github.com/pelevesque/skeletor) - 2D skeletal animation system
* [Sodapop ★26](https://github.com/tesselode/sodapop) - Sprite and animation library
* [Walt ★26](https://github.com/davisdude/Walt) - Animation library
* [Lump ★13 ⏳1Y](https://github.com/sixFingers/lump) - Adobe Flash animation runtime

## Camera

* [Brady ★29](https://github.com/davisdude/Brady) - Camera library with parallax scrolling
* [Editgrid ★23 ⏳2Y](https://github.com/bakpakin/Editgrid) - Gamera and HUMP compatible scaling grid
* [gamera ★111](https://github.com/kikito/gamera) - Camera system
* [hump.camera](http://hump.readthedocs.io/en/latest/camera.html) - Camera library with window locking and smooth camera movement interpolation
* [STALKER-X ★17](https://github.com/SSYGEN/STALKER-X) - STALKER-X is a camera module for LÖVE

* [LoveDebug ★60 ⏳1Y](https://github.com/Ranguna/LOVEDEBUG) - Inline console-like debugger utility
* [lurker ★105 ⏳2Y](https://github.com/rxi/lurker) - Auto-swaps changed Lua files in a running game
* [LÖVE API ★95](https://github.com/love2d-community/love-api) - The complete API documentation of LÖVE in a Lua table

## Drawing
*Drawing specific Libraries, that simplify the Drawing process*

* [Artal ★22](https://github.com/unXedDani/Artal) - A .PSD parsing library for LÖVE
* [Autobatch ★39](https://github.com/rxi/autobatch) - Small LÖVE module to automate the use of SpriteBatches
* [HSLUV ★15 ⏳1Y](https://github.com/hsluv/hsluv-lua) - Lua implementation of HSLuv (a human-friendly alternative to HSL)

* [Möan.lua ★35](https://github.com/twentytwoo/Moan.lua) - Multiple-choice VN-esque messagebox library for LÖVE
* [renderplanet ★9](https://github.com/meric/renderplanet) - realistic orthographic planet rendering
* [Sölar ★1](https://github.com/JanWerder/soelar) - a fairly simple solar system simulator
* [svglover ★17](https://github.com/globalcitizen/svglover) - Library to import and display simple SVGs in LÖVE

## <a name="helper"></a>Helper Libraries
*Game specific Library bundles, that provide reuseable functions*

* [astray ★49 ⏳1Y](https://github.com/SiENcE/astray) - Astray is a lua based maze, room and dungeon generation library for dungeon crawlers and roguelike video games
* [hump ★446](https://github.com/vrld/hump) - Collection of tools for developing games with LÖVE (Gamestates, Timers/Tweens, Vectors, Classes, Signals, Cameras)
* [knife ★127](https://github.com/airstruck/knife) - Collection of useful micro-modules for Lua (Class, State Machines, Bind, Chain, Coroutines, Event, Memoize, Entity, Tests, Timer)
* [lovely-windows ★3](https://github.com/flamendless/lovely-windows) - A Window/Screen Manager Module for virtual windows.
* [love-state-switcher ★5](https://github.com/nekromoff/love-state-switcher) - State switcher class for Lua LÖVE Engine
* [lue ★27](https://github.com/Ulydev/lue) - lue is a LÖVE library that allows you to display hue color effects in your game
* [lume ★229](https://github.com/rxi/lume) - Collection of functions for Lua, geared towards game development
* [rotLove ★116](https://github.com/paulofmandown/rotLove) - Roguelike Toolkit in Love. A Love2D/lua port of rot.js
* [ScreenManager ★16](https://github.com/rm-code/screenmanager) - Screen/State Management for the LÖVE framework
* [shack ★27 ⏳1Y](https://github.com/Ulydev/shack) - shack is a LÖVE library that lets you easily add screen effects such as shake and rotation
* [Simple Tiled Implementation ★319](https://github.com/karai17/Simple-Tiled-Implementation) - Tiled map loader and renderer
* [Vivid ★31 ⏳2Y](https://github.com/WetDesertRock/vivid) - Color math, manipulation and conversion library

## Math
*Math specific Libraries*

* [Cirno's Perfect Math Library ★101](https://github.com/excessive/cpml) - Math/intersection library designed for games
* [delaunay ★59](https://github.com/Yonaba/delaunay) - Delaunay triangulation for convex polygons

* [hump.vector](http://hump.readthedocs.io/en/latest/vector.html) - Powerful 2D vector class
* [Bresenham ★3](https://github.com/rm-code/Bresenham) - Bresenham's line algorithm written in Lua

## <a name="object"></a>Object Orientation
*Object Orientation Libraries that support [Class-Commons ★33 ⏳2Y](https://github.com/bartbes/Class-Commons)*

* [30log ★234](https://github.com/Yonaba/30log) - Minified framework for object-orientation in Lua. It features named (and unnamed) classes, single inheritance and a basic support for mixins
* [classic ★175](https://github.com/rxi/classic) - Tiny class module for Lua. Attempts to stay simple and provide decent performance by avoiding unnecessary over-abstraction
* [hump.class](http://hump.readthedocs.io/en/latest/class.html) - Small, fast class/prototype implementation with multiple inheritance (class-commons)
* [knife.base](https://github.com/airstruck/knife/blob/master/readme/base.md) - Extremely minimal base class providing single inheritance and constructors.
* [middleclass ★892](https://github.com/kikito/middleclass) - Simple OOP library for Lua; has inheritance, metamethods (operators), class variables and weak mixin support (class-commons)

* [ProFi](https://gist.github.com/rm-code/383c98a6af04652ed9f39b7ae536bcc5) - A simple lua profiler that works with LuaJIT and prints a pretty report file in columns
